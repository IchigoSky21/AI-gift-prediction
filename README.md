# 🎁 GIFTo - AI Gift Assistant

![GIFTo Banner](https://img.shields.io/badge/Status-Live_on_Vercel-success?style=for-the-badge&logo=vercel)
![Tech Stack](https://img.shields.io/badge/Tech-Vanilla_JS_%7C_Tailwind_%7C_Serverless-blue?style=for-the-badge)

**GIFTo** adalah asisten cerdas berbasis kecerdasan buatan (AI) yang dirancang untuk membantu Anda menemukan ide kado sempurna untuk orang tersayang, lengkap dengan estimasi harga, alasan pemilihan, dan foto produk asli.

Dibangun dengan antarmuka **Playful Glassmorphism** yang modern, GIFTo memberikan pengalaman pengguna yang cepat, mulus, dan menyenangkan di perangkat apa pun.

🚀 **[Coba Aplikasi GIFTo Langsung di Sini](https://ai-gift-prediction.vercel.app/)**

---

## ✨ Fitur Utama

- **🤖 Smart AI Recommendations:** Diotaki oleh Google Gemini AI untuk meracik ide kado yang sangat spesifik berdasarkan hubungan, acara, usia, hobi, dan *budget*.
- **📸 Real Product Imagery:** Terintegrasi dengan Pexels API untuk menampilkan foto representasi kado beresolusi tinggi secara instan.
- **📱 Progressive Web App (PWA):** Dapat diinstal langsung ke *homescreen* Android, iOS, maupun Desktop (mendukung *offline caching* dasar).
- **🌗 Adaptive Dark Mode:** Antarmuka yang ramah di mata, secara otomatis menyesuaikan dengan preferensi sistem perangkat Anda.
- **🔒 Secure Serverless Architecture:** API Keys disembunyikan dengan aman menggunakan Vercel Serverless Functions (Backend).
- **💼 Fitur Produktivitas:** 
  - Simpan ide ke daftar **Favorit** (❤️).
  - Akses kembali **Riwayat Pencarian** (🕒) sebelumnya.
  - Bagikan ide langsung ke **WhatsApp** (💬).
  - Tautan pencarian langsung ke **Tokopedia & Shopee** (🛒).

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, Vanilla JavaScript, Tailwind CSS (via CDN).
- **Backend / API:** Node.js (Vercel Serverless Functions).
- **AI Engine:** Google Gemini (Generative Language API).
- **Image Engine:** Pexels API.
- **Deployment:** Vercel.
